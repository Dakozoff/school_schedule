# school_schedule

# Установка
**Версия python не меньше 3.9**

  1. Установка библиотек
     
      ```python3 -m pip install -r requirements.txt```
  3. Настройка бота
     
       ```В файле config.py можно изменить кол-во максимальных уроков учителя за день```
  4. Запуск
     
       ```python3 main.py```
